# 地支合冲刑害关系全表

> 所有生克以五行生克为基础，以下关系为叠加条件。同时成立时综合分析，不可孤立判断。

---

## 一、地支六合（爻层面）

| 六合配对 | 合化 |
|----------|------|
| 子丑合 | 土 |
| 寅亥合 | 木 |
| 卯戌合 | 火 |
| 辰酉合 | 金 |
| 巳申合 | 水 |
| 午未合 | 土 |

**断点**：
- 爻爻相合 → 彼此牵绊，作用暂缓，逢冲方解
- 日月合爻 → 该爻被合绊，暂不能生克他爻
- 动爻与变爻相合 → 动而逢合，事有羁绊
- 用神逢合 → 吉则拖延，凶则暂不发作
- **合处逢冲** → 先合后散，事成复败
- **冲处逢合** → 先散后合，冲突得调

## 二、地支三合局

三合局是六爻中最强大的联动关系，力量远超单爻生克。只有动爻（含暗动）与日月方可组成。

| 三合局 | 属性 | 长生→帝旺→墓库 |
|--------|------|-----------------|
| 申子辰 | 水局 | 申(长生) 子(帝旺) 辰(墓库) |
| 亥卯未 | 木局 | 亥(长生) 卯(帝旺) 未(墓库) |
| 寅午戌 | 火局 | 寅(长生) 午(帝旺) 戌(墓库) |
| 巳酉丑 | 金局 | 巳(长生) 酉(帝旺) 丑(墓库) |

**成局条件**：三字全现+至少一字发动（或日月占一字）。二字半合需有动爻。

**断点**：
- 三合原神局（生用神）→ 多方来助，大吉
- 三合忌神局（克用神）→ 四面楚歌，大凶
- 三合用神局（同五行）→ 用神极旺
- 三合局一字被冲散 → 合局破裂，吉凶皆解

## 三、地支三会局

| 三会局 | 属性 | 组成 |
|--------|------|------|
| 寅卯辰 | 东方木 | 寅+卯+辰 |
| 巳午未 | 南方火 | 巳+午+未 |
| 申酉戌 | 西方金 | 申+酉+戌 |
| 亥子丑 | 北方水 | 亥+子+丑 |

**成局条件（严格）**：
1. 三方地支**全现**于卦中（含本卦爻、变爻、伏神）
2. 三方**均**在动爻或变爻中（静爻不计数，日月可替补一字）
3. 至少一字发动

> ⛔ 三会局极为稀有。若三方不全在动/变中（如仅二字动、一字为静爻），则**降级为「三会之势」**——力量仍强（远超单爻生克），但非严格成局，不宜用「碾压」等绝对化措辞。

**降级评估（必须执行）**：
- 认定存疑时，主动评估「降级是否改变核心断语」。
- 若降级后核心结论不变 → 注明「三会未严格成局，但X+Y+Z等独立信号已足以支撑断语」
- 若降级后结论动摇 → 重新审断，独立信号是否仍成立

**认定报告（须在第三步输出中显式呈现）**：
```
三会局认定：
- 三方：[X]（[动/静/变/日月]）、[Y]（[状态]）、[Z]（[状态]）
- 判定：[严格成局 / 三会之势 / 不成立]
- 降级评估：[降级不改变断语，因... / 需重新审断...]
```

三会局一旦严格成局，方向性极强，无论生克势不可挡。三会之势虽降级，合力仍远超单爻生克。

## 四、地支六冲（爻层面）

| 六冲配对 |
|----------|
| 子午冲 |
| 丑未冲 |
| 寅申冲 |
| 卯酉冲 |
| 辰戌冲 |
| 巳亥冲 |

**爻爻相冲 ≠ 六冲卦**：六冲卦（如乾为天）是卦的结构属性；爻爻相冲是局部关系。

**断点**：
- 用神逢冲 → 事有变动
- 世应相冲 → 双方对立
- 动爻逢月冲 → 月破
- 静爻旺相逢日冲 → 暗动

## 五、地支相刑

| 刑类 | 配对 | 含义 |
|------|------|------|
| 无礼之刑 | 子卯刑 | 以下犯上，无理取闹 |
| 无恩之刑 | 寅巳、巳申、寅申 | 忘恩负义，反目成仇 |
| 恃势之刑 | 丑戌、戌未、丑未 | 仗势欺人，恃强凌弱 |
| 自刑 | 辰辰、午午、酉酉、亥亥 | 作茧自缚，自寻烦恼 |

断点：刑在动爻→过程摩擦；刑在用神→法律/道德风险；刑在世爻→决策失误/心理矛盾。刑属于隐性问题，不决定吉凶方向但持续磨人。
