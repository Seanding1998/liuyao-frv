> 此文件为 SKILL.md 的详细执行补充。SKILL.md 定义流程骨架（门禁+输出+检查点），此文件提供每步的具体操作表和规则。进入每一步前按 SKILL.md 的加载规则载入对应章节。

## 16条核心原则（全文）

1. **用神为纲**：一切分析围绕用神，不看用神就是瞎断。
2. **旺衰为本**：用神自身旺衰是第一位的，动爻生克是第二位的。
3. **动爻是故事的导演**：静卦直接看用神旺衰 + 世应关系。
4. **日月如天**：月建为一月之主，日辰为一日之主，生克权高于卦内爻。
5. **卦不妄成**：每一个动爻、旬空月破都有含义。
6. **实事求是**：凶就说凶，吉就说吉；凶中指救应，吉中防暗藏。
7. **不知为不知**：罕见格局坦诚说明，不编造。
8. **伏神必查**：用神不现立即查伏神。
9. **冲合分虚实**：暗动/日破/冲散细辨，六合/六冲逐爻精判。
10. **三合为大**：三合局一旦成局，力量凌驾单爻生克之上。
11. **刑害为隐**：遇刑则纠葛，遇害则暗算——不决吉凶方向但揭示暗礁。
12. **贪忘有先后**：优先级 合 > 冲 > 刑 > 害 > 生 > 克。
13. **动则不空**：发动之爻不以其旬空论无力，动即有实。
14. **动不为破**：发动之爻不以其月破论无用，破中能动。
15. **旺不为空**：旺相之爻虽旬空不为真空。
16. **化出定结局**：动爻化出之变爻决定事情最终收场。

### 用神定位要点

1. 遍历卦中所有爻的 `liu_qin` 字段，查找与主用神六亲匹配的爻。
2. 若匹配 → 用神明现，记录爻位和地支。

4. 若完全不匹配 → 用神不现于卦面，**必须标记**后续执行第3.5步。

## 第二节：判定用神旺衰（第二步用）

### 月建判定（基础，与日辰等权）

> ⛔ 日月如天，月建与日辰权重对等（均为 ±1）。六爻测的是当下的能量状态，月与日同为此时此地的「气」，不分轻重。

| 月建与用神关系 | 结果 | 分值 |
|---------------|------|------|
| 月建同五行（当令） | **旺** | +1 |
| 月建生扶用神 | **相** | +1 |
| 月建克用神 | **死** | -1 |
| 用神生月建 | 休 | -1 |
| 用神克月建 | 囚 | -1 |

> ⛔ 若月建与用神地支存在六合/半合/三合关系，克的负面效应被合保护而抵消，视为「克中有护」，该条克不扣分（死 → 0）。仅当合、冲、克同时存在时，按合>冲>克优先级取最高者。

### 日辰影响（叠加）

- 日辰生扶/比和 → +1
- 日辰克 → -1
- 日辰冲用神 → 必须区分「暗动」与「日破」
  - 用神旺相/旬空 + 日冲 → **暗动**，不计负分，反得短期有力
  - 用神休囚 + 日冲 → **日破**（-2，结构破损，与月破对等）

### 暗动 vs 日破速判

| 条件 | 结论 | 含义 |
|------|------|------|
| 静爻旺相 + 日辰冲 | **暗动** | 虽静实动，有力，短期有效 |
| 静爻休囚 + 日辰冲 | **日破** | 被冲破，无力 |
| 静爻旬空 + 日辰冲 | **暗动**（冲空则实） | 反得有力 |
| 静爻月破 + 日辰冲 | **日破**（破上加破） | 彻底无用 |
| 动爻 + 日辰冲 | 日冲散 | 力量减半，非全废 |

> 详细规则：`references/dong-bian-fa-ze.md` 第一、二节

### 特殊状态叠加

- 旬空（kong_wang:true）：暂无力，但需考虑「旺不为空」「动则不空」
- 月破（与 month_branch 相冲）：根基动摇(-2)
- 入墓（用神地支为日墓或动墓）：受限迟缓

### 综合判定（四档必选一）

> 日月分值叠加后计算总分，再结合特殊状态定档：

| 综合状态 | 条件 |
|----------|------|
| **旺相有力** | 总分 ≥ +2（如月旺+1 + 日比和+1），无严重特殊状态 |
| **中和可用** | 总分在 0~+1 之间（日月抵消），或总分 -1 但有日辰生扶/合的关系护持 |
| **休囚无力** | 总分 ≤ -2（月日双克），且无救应 |
| **空破无用** | 真空（休囚总分≤-1 且旬空，无合无生）或真破（休囚总分≤-1 且月破+日破，无救应）

## 第三节：追踪动变路径（第三步用）

### 3.1 动爻角色判定

| 动爻与用神关系 | 角色 |
|---------------|------|
| 动爻生用神 | 原神 |
| 动爻克用神 | 忌神 |
| 与用神同五行 | 兄弟 |
| 被用神生 | 泄气 |
| 被用神克 | 财源 |

### 3.2 变化性质判定

逐项检查动爻的 `bian_yao`：
- **回头生**：变爻生本爻 → 后劲充足
- **回头克**：变爻克本爻 → 原计划受阻
- **化进神**：同五行顺行上位（寅→卯、巳→午、申→酉、亥→子、辰/未/丑→戌）
- **化退神**：同五行逆行下位（卯→寅、午→巳、酉→申、子→亥、戌→未/辰/丑）
- **化空**：变爻地支在 kong_wang 列表中
- **化破**：变爻地支与 month_branch 相冲
- **化墓/绝/长生/帝旺**：依十二长生表判定（见 `references/dong-bian-fa-ze.md` 第九节）
- **化合**：动爻与变爻地支相合 → 被合绊

> ⛔ 化进/退和十二长生必须查 `references/dong-bian-fa-ze.md` 第三节和第九节，不得凭印象。

### 3.3 多爻联动规则

| 模式 | 判定 | 含义 |
|------|------|------|
| 连环相生 | 多爻依次相生至用神 | 大吉 |
| 忌神被制 | 忌神被另一动爻克住 | 反凶为吉 |
| 救应 | 忌神克用→原神来救 | 先凶后吉 |
| 三合局 | 三动爻地支成申子辰/亥卯未/寅午戌/巳酉丑 | ⚠️ 凌驾单爻生克 |
| 贪生忘克 | 一爻既生某又克某，先生后克 | 暂时忘克 |
| 贪合忘克 | 克用神之爻去合另一爻 | 暂时忘克 |
| 仇神 | 克原神之爻 | 断用神后援 |

**多爻联动优先级**：合 > 冲 > 刑 > 害 > 生 > 克。三合局一旦成局，力量超过任何单爻生克。

## 第 3.5 节：用神伏藏与飞神分析（第3.5步用）

### 飞伏生克五态

| 飞伏关系 | 名称 | 含义 |
|----------|------|------|
| 飞神生伏神 | 飞来生伏 | 大吉，伏神得长生 |
| 伏神生飞神 | 伏去生飞 | 泄气，需主动 |
| 飞神克伏神 | 飞来克伏 | 大凶，被压制 |
| 伏神克飞神 | 伏去克飞 | 出暴，费力有斩获 |
| 飞伏比和 | 飞伏比和 | 平，需时机 |

### 出伏四条件

1. **冲飞之日**：日辰冲飞神 → 伏神得出
2. **值伏之日**：日辰与伏神地支相同 → 伏神当值
3. **飞神空破**：飞神旬空/月破/日破 → 压制解除
4. **伏神旺相**：伏神得月建生扶 → 虽伏有力

### 口诀

- 「伏居空地，事不待冲」——飞神空亡，伏神自动出头
- 「伏神空亡，万事难成」——伏神自身旬空，即使出伏也无用
- 「伏来生飞为泄气，飞去生伏得长生」

> 飞伏口诀与日月入卦、卦身详细规则见 `references/fushi-riyue-guashen.md`

### B. 一致性交叉验证规则

1. **旺衰 ↔ 应期**：用神"旺相有力"，应期却是"待旺相之日"？→ 矛盾，修正应期。
2. **旺衰 ↔ 断语定性**：用神"空破无用"，断语却定性为"吉"？→ 必须有极强救应，否则矛盾。
3. **动变 ↔ 断语**：忌神发动克用神且无救应，断语却定性为"吉"？→ 矛盾。
4. **世应 ↔ 断语**：应克世（外部阻力大），断语却给出顺畅建议？→ 矛盾。
5. **伏神 ↔ 用神**：用神明现但跳到了3.5步，或用神不现但没执行3.5步？→ 步骤错误。
6. **三合 ↔ 单爻**：识别出三合局，但断语中只分析了单爻生克？→ 遗漏（三合为大）。
7. **静卦 ↔ 动变**：卦无动爻但写了动变分析，或有动爻但只写"静卦"？→ 步骤错误。
