# 伏神飞神·日月入卦·卦身

> 对应主 Skill 第 3.5 步和高级辅助判断。

---

## 一、伏神与飞神

当卦中没有明现用神时，必须查伏神。飞神即覆盖在伏神之上的本卦之爻。

### 飞伏生克五态

| 飞伏关系 | 名称 | 含义 |
|----------|------|------|
| 飞神生伏神 | **飞来生伏** | 大吉，伏神得长生，用之有力 |
| 伏神生飞神 | **伏去生飞** | 泄气，伏神力量外泄，事需主动 |
| 飞神克伏神 | **飞来克伏** | 大凶，伏神被压制，难以出头 |
| 伏神克飞神 | **伏去克飞** | 出暴，冲出即有斩获，但费力 |
| 飞伏比和 | **飞伏比和** | 平，事可成但需时机 |

### 伏神出伏四条件

1. **冲飞之日**：日辰冲飞神 → 伏神得出
2. **值伏之日**：日辰与伏神地支相同 → 伏神当值
3. **飞神空破**：飞神旬空/月破/日破 → 压制解除，伏神自动出头
4. **伏神旺相**：伏神得月建生扶 → 虽伏而有力，待时而发

### 口诀

- 「伏居空地，事不待冲」——飞神空亡，伏神自动出头
- 「伏神空亡，万事难成」——伏神自身旬空，即使出伏也无用
- 「伏来生飞为泄气，飞去生伏得长生」

## 二、日月入卦

当用神/原神/忌神的地支等于日辰或月建时，该爻得到「天时」加持：

| 情况 | 含义 | 影响 |
|------|------|------|
| 用神临月建 | 用神与月建同地支 | 用神极旺，当月权重最高 |
| 用神临日辰 | 用神与日辰同地支 | 用神当值，当日有力 |
| 忌神临月建 | 忌神与月建同地支 | 忌神得时，当月凶甚 |
| 忌神临日辰 | 忌神与日辰同地支 | 忌神当值，当日凶显 |
| 原神临月建 | 原神与月建同地支 | 原神有力，生用可靠 |

> 诀：爻值日月，权倾一时。吉凶因之加倍，过则恢复常态。
