## 二、五行相生（顺生）

```
木 → 火 → 土 → 金 → 水 → 木
```

| 生者 | 被生者 | 口诀 | 含义 |
|------|--------|------|------|
| 木 | 火 | 木生火 | 木燃生火 |
| 火 | 土 | 火生土 | 火烬归土 |
| 土 | 金 | 土生金 | 土中藏金 |
| 金 | 水 | 金生水 | 金熔为水 |
| 水 | 木 | 水生木 | 水润生木 |

**在六爻中的应用**：
- 生我者为父母（生我者，养我者也）
- 我生者为子孙（我生者，我所出也）
- 原神即生用神之爻（如用神为妻财，子孙爻即原神）
- 月建/日辰生用神 → 用神得生，旺

## 三、五行相克（顺克）

```
木 → 土 → 水 → 火 → 金 → 木
```

| 克者 | 被克者 | 口诀 | 含义 |
|------|--------|------|------|
| 木 | 土 | 木克土 | 树根固土 |
| 土 | 水 | 土克水 | 堤坝挡水 |
| 水 | 火 | 水克火 | 水能灭火 |
| 火 | 金 | 火克金 | 火熔金石 |
| 金 | 木 | 金克木 | 斧砍树木 |

**在六爻中的应用**：
- 克我者为官鬼（克我者，制约我也）
- 我克者为妻财（我克者，我掌控也）
- 忌神即克用神之爻（如用神为妻财，兄弟爻即忌神）
- 月建/日辰克用神 → 用神被克，衰
