## 一、车辆

1. 车辆以**父母爻**为用神。
2. 申酉二金爻车辆受日建克多为乳白色。
3. 寅卯木爻车辆多是深绿、油绿、天蓝色（解放牌、油田用卡车）。
4. 巳午火日月建均红色车（新）。申酉二爻为父爻，日月有巳火，车头为红色（如拖拉机）。
5. 子水父爻受申酉日月生者，均黑色轿车。
6. 土爻为木黄色，可据日月生克灵活掌握花间色。**日建最大，月建较大，动爻最小**。
7. 受日月生者为新车，合日月也是新车，受冲是车破或有毛病。
8. 受动爻生是翻新车。卯木动化子水、巳火动化寅木相生 → 车改得不好（因相刑）。
9. 日月动爻刑合父母爻：
   - 五爻 → 方向盘不灵
   - 四爻 → 内燃机不行
   - 三爻被冲克 → 油箱不好
   - 二爻 → 车轮有毛病
   - 上六爻刑合 → 车头不灵
   - 朱雀被合 → 喇叭不行
   - 父爻被日月冲 → 刹车不灵（阳支左打不灵，阴支右打不灵）
10. 三四爻酉金临父母（尤其三爻），被卯木冲克 → 漏水漏油修不好；申酉金下有亥水 → 漏油不止。
11. **官鬼发动** → 车辆易坏。官父同动 → 车辆肇事。官伤兄弟、父伤子孙 → 车祸大，死人。父爻土动化午回头生 → 车在阴历五、九月出事，在西北东南交通线上。
12. 父母土爻，寅卯木财爻发动 → 碰树。金父爻被卯木冲动 → 碰树。金动冲寅卯木 → 在桥上出事、碰树干、落河。父与父相冲 → 两车相撞。白虎临木较严重，白虎临土也凶险，白虎临火伤灾较轻，虎临子孙有惊无失，临朱雀相撞要起火。

## 二、求财

| 要诀 | 详解 |
|------|------|
| 财爻克世 | 得财快；克世谨防父母灾殃 |
| 财爻生世 | 得财慢；挣钱后可能得病或有官司（官爻安静则无事） |
| 安静财爻逢冲 | 财到 |
| 月建临财 | 本月得财 |
| 日建临财 | 天天有财 |
| 财不上卦 | 若属相、世爻与财作合，或其墓库，钱财多 |
| 财爻持世 + 子孙动生，父爻安静 | 利润滚滚 |
| 财爻持世临日月，父动子静 | 只能做一次生意 |

**求财方位**：到子孙位、财位求财。销货到兄弟位要赊销，临父爻要削价处理，也可到官位求财（制兄弟）。世爻与属相去临生合局方位销货（三合同）。

**财爻五行看行业**：
- 亥子水财爻 → 杀猪宰羊、水产品、调料、干菜水果
- 寅午戌财爻 → 饭店、烧窑、电厂、油库、金属器材、电器
- 巳酉丑财爻 → 金属器材、电器、建材、玻璃
- 亥卯未财爻 → 工艺品、维修装潢、植物栽培、木器家具、琴棋书画
- 勾陈临财临世临官 → 基建或房地产开发
- 寅卯木财爻 → 烟酒、电工（午火）
- 朱雀 → 以说话挣钱（经纪人、教书）
- 青龙 → 演员、理发美容
- 空官临世 → 无本求财（相面、算卦、江湖术士）
- 玄武 → 干私活、走私

**子孙爻（货源/产品）**：无子孙上卦 → 皮包公司。子孙临空带朱雀 → 用口挣钱（经纪人）。子孙受刑 → 产品质量不好要亏本处理。子孙化退 → 价落；化进 → 涨价。子孙被日月合 → 不易销，逢冲日可卖。

**合伙与禁忌**：
- 兄弟持世不求财，找子孙或父母合伙尚可
- 合伙必须一局或小六合，否则不合
- 凡寅巳申三刑 → 经济问题要出事；缺一空亡 → 合同不真货物不实
- 求财日期：子孙临旺、财爻五行到位时
- 财爻过多 → 墓库收藏；入墓之财逢冲日月得

## 三、官讼

### 基本判断

| 情形 | 断语 |
|------|------|
| 得天水讼卦 | 官司终不能成讼，私人调解较好 |
| 世爻临父母不被冲不受空亡 | 有理在握 |
| 应爻临财动 | 有理也打不赢（财克父母） |
| 应爻是官 | 肯定能判赢 |
| 父母空亡 | 无人受理、理由不足 |
| 父母受刑 | 理由不足，多数驳倒 |
| 父母被合 | 有理不会讲 |
| 父母入墓 | 无话可讲 |
| 父母化退 | 有撤诉之象 |
| 父母入日墓 | 判决慢；入月墓 → 出月判；化动墓 → 逢冲之年判 |

### 世爻角色的影响

| 世爻 | 影响 |
|------|------|
| 世临兄爻 | 虽受父母生却要挨官打，无理打不赢；入父墓/财墓 → 入狱 |
| 世临财爻 | 打官司有利（财是官的原神） |
| 世临兄爻，官鬼动 | 最好撤诉 |
| 兄爻发动 | 耗财花钱 |
| 子孙持世 | 私自和解 |

### 审理层级

- 官爻在初爻 → 当地解决
- 二五爻 → 上级解决
- 临日月 → 靠上级判决
- 两官两父 → 两次官司、二级判决
- 官爻多 → 多方审理

### 判决与刑狱

- 官爻化退 → 不愿受理、消极怠慢
- 属相入虎墓或被白虎生合 → 要判刑
- 勾陈临属相墓 → 要判刑
- 寅午戌年犯罪 → 申子辰年出；合年进去 → 冲年出

## 四、婚姻

### 男女分看

**女方**：
| 情形 | 断语 |
|------|------|
| 小女孩临财爻 | 不听父母话（财生官） |
| 临兄爻 | 一般被男方退亲（官克兄） |
| 临子孙 | 有孩，性格活泼外向，带龙会喝酒，带玄武暧昧 |
| 财子临白虎 | 流过产 |
| 子孙持世 | 男人必死（克夫） |
| 遇衰官 | 男体弱终身无为 |
| 遇旺官 | 男人成材 |
| 官空绝破 | 克夫伤子 |

**男方**：
| 情形 | 断语 |
|------|------|
| 临兄爻 | 恋爱娶妻心情迫切，家境贫寒，逢旺年克妻 |
| 临子孙持世 | 乐观好玩，舍得为爱花钱，多次恋爱不成功 |
| 财受克 | 晚婚 |
| 遇兄世 | 必伤原配 |
| 遇财动化沐浴克世 | 有奸杀之相 |

### 世应关系

- 世应均阳爻 → 一谈即崩
- 世应均阴爻 → 对方慢慢冷淡
- 世应俱空 → 成而终散
- 应化退 → 对方退婚；世化退 → 自己退婚
- 世应刑冲克 → 生离死别
- 旺世克应 → 嫌贫爱富
- 世应相错 → 妻夺夫权
- 男官世女财应 → 婚姻美满
- 男财世妇官应 → 阴阳反错，婚后被管制
- 子午卯酉或爻旺 → 长得漂亮

### 卦象格局

- 逢六冲卦 → 必散（不过三年）
- 六合卦 → 易美满
- 六爻安静 → 家庭和睦
- 两头冲 → 必散；中间合 → 能成
- 内外卦两官两财 → 再婚之兆
- 财官重叠 → 各有所欢
- 兄财互化 → 婚不吉
- 兄化兄 → 终不成就
- 兄世虎动 → 必然伤妻

### 离婚判断

- 双方有一个兄弟爻 → 有离散信息
- 兄爻与子孙生合 → 为孩子离婚动摇
- 女人问卦，兄弟持世 → 丈夫厌烦女方
- 女人问卦，子孙持世 → 女的烦丈夫
- 空亡 → 没下决心离
- 日月爻为父母或父母爻发动 → 男方已上诉法庭
- 被父母合住 → 感情不好也不离

### 婚期与外遇

- 临青龙的爻位 → 婚期（如辰爻五一，丑爻春节，卯木二月）
- 玄武发动化合 / 财动化官 / 官动化财 → 私婚
- 看青龙方位入墓处 → 外遇方位
- 财官重叠 → 各有所欢
- 月令合旺财加玄武 → 女必重婚

## 五、疾病

### 爻位对应人体

| 爻位 | 对应部位 |
|------|----------|
| 初爻 | 足 |
| 二爻 | 膝、关节 |
| 三爻 | 胃、肾、脾、胰、泌尿系统、妇科 |
| 四爻 | 乳腺、心脏、背、肝、胆、纵隔 |
| 五爻 | 脖子、肩周、颈椎、咽喉、气管 |
| 六爻 | 头部、耳鸣、头晕、血压 |

### 八卦对应人体

艮手、巽股、离目、坤腹、乾头、兑口、坎耳、震足。

### 专项速查

**头部**：上爻临白虎 → 血液病变（贫血、高低血压、头痛）。上爻临戌土，卦中有卯木，巳火不生戌土 → 贫血、头晕眼花、四肢无力。上爻临巳火兄爻，卯木父爻生助 → 偏头痛。

**失眠**：酉金持世，卯木在六爻 → 失眠。寅申相冲 → 寅时就起，逢冲就起，逢合就眠。

**咽喉**：五爻被刑合 → 咽喉不好/气管不好/颈椎病。子月子日占子水临五爻 → 咽喉干燥。

**心脏**：四爻未土动化午火合 → 冠心病/供血不足。四爻临戌土有卯木合 → 胸闷/乳腺炎。丑戌相刑 → 心脏不好。

**肠胃**：四爻酉金卯酉相冲 → 拉肚子、直肠癌。三爻酉金被月建冲克 → 胃不好、腰痛、间盘脱出。财爻在三爻 → 饮食不多；空 → 吃饱就难受；被冲 → 反胃。

**肝肾**：寅卯木在四爻或卦中木爻动 → 肝胆病变。申金化巳火、巳申合 → 心脏不好。

**妇科/泌尿**：三爻子水发动入辰库 → 生育力差、子宫狭窄不孕。二爻临巳火 → 妇女小腹痛涨满。初爻辰土化卯回头克临白虎 → 腹部开刀。

**手术**：三四爻临白虎 → 手术（老阳过去开刀，老阴今后开刀）。白虎动化子孙 → 手术成功；化鬼/化兄/化财 → 不成功。

**医药**：子孙旺相 → 吃药有效，病可治。官鬼旺相，子孙被冲克或不上卦 → 绝症药无效。坤/困/屯/蹇 → 多为不能动，躺着。

**分左右**：寅木左腿，卯木右腿。阳支 → 病在左边；阴支 → 病在右边。

## 六、调转工作与升学

### 调转工作

1. 首看**二爻、五爻**，再看日月生扶。
2. 父爻临身生世 → 调动可以，不用再找贵人。
3. 喜静爻父母 → 到冲时可调成。
4. 冲属相年月 → 多为调动工作之时。
5. 属相合官、合父母（三合及六合内）→ 调提皆有望。
6. 属相世爻在**三合局**内或在日月二五爻位 → 子孙也能帮忙。
7. 找三合局内熟人 → 如寅卯属相找亥猪（姓朱）、属兔形成亥卯未木局。
8. 父爻被日月合/入墓/不上卦 → 调不成。

**方位利益**：子孙方位 → 安逸没活干；财位 → 有钱工资高；官位 → 提升快；在财地不提，转调官地 → 马上升官。

### 升学

| 情形 | 断语 |
|------|------|
| 父母持世被生合 | 可考上 |
| 合住父母 | 很用功但临场发挥不好 |
| 父母合上月日建 | 多考不好，逢冲之年能考上 |
| 父母化回头生合 | 能考上 |
| 官父发动或化父母回头生合 | 已被录取 |
| 父爻生属相/生世爻 | 用神方位即要考的学校 |
| 临子孙、财爻 | 多考不上自费上学 |
| 子孙化财、财化子孙 | 委培生或替考 |
| 财爻在六爻入墓或相合 | 考不上，进考场就晕 |
| 兄弟爻 | 到父母爻方位可考上 |
| 财官相生 | 通过关系读书 |

## 七、家宅

### 宅人关系

- **二爻为宅，五爻为人**。宅生人 → 吉；宅克人 → 凶，速搬家。人克宅 → 尚可居。
- 官鬼 → 楼房；父爻 → 平房。旺者为新。

### 地下之物

- 二爻有官鬼 → 宅基有坟。官爻在外卦 → 坟地较远。
- 官鬼在什么位置或入墓地点 → 坟的方位。
- 不受冲克的土爻 → 地；受日月动爻冲 → 此地已被掘土。
- 辰戌丑未动化回头克 → 被挖坑；回头生 → 填平的坑。
- 鬼爻临合爻+白虎 → 非正常死亡（杀头、战争、车祸、血癌等）。

### 水路

| 地支 | 方位与含义 |
|------|------------|
| 辰土在二爻临青龙 | 东边有河/坑/大水沟（若卦有酉合辰 → 西方或西南） |
| 亥子水旺 | 大水；休囚 → 小水、下水道、水管 |
| 亥水 | 西北有水/坑/洗澡塘 |
| 子水 | 北方有下水道；旺者为河；在初爻 → 宅南有水 |
| 申酉金 | 水的发源地，西南向东北流 |
| 水爻临父母 | 家庭求孩子必改下水道 |
| 水临官爻克壮年 | 年轻小伙子易有病 |

### 门窗道路

- 冲兄爻 → 宅的门路（最吉，因为冲兄生孩子、冲鬼孝父母）。
- 兄爻多发动 → 破财克妻，应改造门。
- 辰戌 → 东西路；卯酉 → 东西道；子午 → 南北路；丑未 → 南北路/偏东北西南。
- 寅巳申三刑 → 弯曲路口；丑未戌三刑 → 胡同不正街道不直。
- 旺相 → 路长；休囚 → 路短。

### 神佛

- 鬼爻临金 → 有桥/有庙/有神位。
- 金在四五六爻 → 菩萨；在三爻 → 仙鬼。
- 日月动爻生 → 仙；日月动爻冲 → 鬼。
- 菩萨金爻与世爻属相相生 → 吉；克世克属相 → 不吉。
- 午火动克金或寅木午火爻动 → 不信神。
- 敬神 → 家中生旺加倍。
- 财神 → 在出门右边，面向北，高一点放。
