# 动变法则详解

> 对应主 Skill 第二步和第三步的详细规则。

---

## 一、暗动与日破

日辰冲用神/静爻时，需严格区分：

| 条件 | 结论 | 含义 |
|------|------|------|
| 静爻旺相 + 日辰冲 | **暗动** | 虽静实动，有力作用，可生克他爻 |
| 静爻休囚 + 日辰冲 | **日破** | 被日辰冲破，无力生克，如无用之物 |
| 静爻旬空 + 日辰冲 | **暗动**（冲空则实） | 空被冲实，反而有力 |
| 静爻月破 + 日辰冲 | **日破**（破上加破） | 彻底无用 |
| 动爻 + 日辰冲 | **日冲散** | 力量减半但并非全废 |

暗动爻特性：可生克他爻似明动，但力只在当下，短期有效，过日则衰，且不能化出变爻。

## 二、旬空与月破

**旬空**（区分日空/月空）：
- **日空**：爻地支在日柱旬中 → 日辰层面暂无力，出旬或值日则实
- **月空**：爻地支在月柱旬中 → 月建层面暂无力，出月或值月则实
- 日空月空可能相同（日月同旬），也可能不同——各算各的，不叠加
- 通用法则：吉凶减半，待出空。但「旺不为空」「动则不空」优先

**月破**：用神地支与月建相冲 → 当前破损（-2，根基动摇，与日破对等）

**入墓**：用神地支入日墓或动墓 → 受限、迟缓

## 三、化进化退详表

进神：动爻地支 → 变爻地支（同五行，顺行上位）

| 五行 | 进神方向 | 实例 |
|------|----------|------|
| 木 | 寅→卯 | 寅木动化卯木，进神 |
| 火 | 巳→午 | 巳火动化午火，进神 |
| 金 | 申→酉 | 申金动化酉金，进神 |
| 水 | 亥→子 | 亥水动化子水，进神 |
| 土 | 辰→戌、未→戌、丑→戌 | 土进神较特殊，辰/未/丑皆可化戌为进 |

退神：动爻地支 → 变爻地支（同五行，逆行下位）

| 五行 | 退神方向 | 实例 |
|------|----------|------|
| 木 | 卯→寅 | 卯木动化寅木，退神 |
| 火 | 午→巳 | 午火动化巳火，退神 |
| 金 | 酉→申 | 酉金动化申金，退神 |
| 水 | 子→亥 | 子水动化亥水，退神 |
| 土 | 戌→未、戌→辰、戌→丑 | 土退神从戌退 |

> 口诀：进神如木之生长，退神如水之回流。进则一日强过一日，退则日渐消退。

## 四、空破在动爻中的特殊法则

| 法则 | 内容 | 说明 |
|------|------|------|
| **动则不空** | 动爻即使旬空，发动之力仍有效 | 动是实在行动，空只表示出空后正式应验 |
| **动不为破** | 动爻即使月破，发动之力仍存在 | 月破只是根基受损，动表示行动正在进行 |
| **旺不为空** | 旺相之爻虽旬空，不以为真空 | 旺相本身有力，空为暂时性；休囚又空才是真空 |
| **动而化空** | 动爻化出的变爻旬空 | 事做完后落空，结局不如预期 |
| **空化空** | 旬空之爻发动，化出之爻也旬空 | 两头落空，大事难成。待出空日月方可论吉凶 |
| **动而化破** | 动爻化出的变爻月破 | 动则有力但最终破败（先成后败） |
| **动散** | 动爻被日辰冲 | 力量减半但并非全废 |

## 五、贪生忘克与贪合忘克

**贪生忘克**：一爻同时生某爻、克某爻时，先生后克。若该爻先去生另一爻，则暂时忘掉克的关系。
> 例：寅木动，本应克戌土用神，但寅木先生了午火，便「贪生忘克」，戌土暂时无碍。

**贪合忘克**：一爻去合另一爻时，该爻被合绊，暂时失去生克能力。
> 例：子水动，本应克巳火用神，但子水与丑爻相合，便「贪合忘克」，巳火暂时安全。

**优先级排序**：合 > 冲 > 刑 > 害 > 生 > 克

## 六、日辰合绊的深度解读

> 当爻（尤其是动爻或用神）被日辰合住时，不可仅止于「被合绊」三字。按以下三层递进解读。

**第一层 — 现象**：确认谁合谁（日辰 X 合爻 Y），断「该爻暂时不能生克他爻」。

**第二层 — 时效**：日辰即「今天」，合绊即「被此刻锁住」。始于今日、终于冲开日（如午未合，子日解）。期间该爻如陷泥潭，有力使不出。

**第三层 — 寓意**（按角色区分）：
| 日辰合… | 含义 |
|----------|------|
| 用神 | 对方/事被今天的外部因素拖住，非其本意 |
| 原神 | 援手被此刻的时间卡住，短期待释放 |
| 忌神 | 坏因素暂时不来，但出冲日将至 |
| 世爻 | 你自己被今天困住了，越想动越动不了 |

**关键断语公式**：「日辰合 X 爻 = 不是 X 不想动，是今天的外部条件锁住了 X。冲开日释放。」

> 此节与第五节「贪合忘克」互补：第五节侧重爻爻相合，此节侧重**日月合爻**的时效解读。

## 七、独发与独静

- **独发**：仅一爻发动。此爻是卦的核心，一切分析从此爻展开。独发为原神→一力来助；独发为忌神→单刀直入。
- **独静**：五爻皆动，仅一爻不动。不动之爻是唯一稳定因素，以静爻为纲。
- **六爻全静**：静卦只看用神旺衰+世应关系。用神旺则吉，衰则凶。

## 八、仇神

| 角色 | 定义 | 作用 |
|------|------|------|
| 原神 | 生用神之爻 | 助用 |
| 忌神 | 克用神之爻 | 攻用 |
| **仇神** | **克原神之爻** | 断用神后援 |

- 原神动来生用神，看似大吉→若仇神同动克原神，生用之力被截断
- 忌神动来克用神，原神来救→若仇神克住原神，救兵被挡

## 九、地支十二长生速查

十二长生用于判断动爻化出之变爻的质量：

| 阶段 | 含义 | 断卦要点 |
|------|------|----------|
| 长生 | 初生、开始 | 化长生→事刚启动，后劲足 |
| 沐浴 | 柔弱、桃花 | 化沐浴→事涉感情/不成熟 |
| 冠带 | 成长、成形 | 化冠带→渐入正轨 |
| 临官 | 就职、到位 | 化临官→到岗就位 |
| 帝旺 | 巅峰、极致 | 化帝旺→顶峰，盛极易衰 |
| 衰 | 衰退 | 化衰→力有不足 |
| 病 | 生病、缺陷 | 化病→事有病态/隐患 |
| 死 | 终结、僵化 | 化死→事无生气 |
| 墓 | 入库、困顿 | 化墓→受困，待冲 |
| 绝 | 断绝、消散 | 化绝→中断，后续无力 |
| 胎 | 孕育、暗中 | 化胎→在酝酿 |
| 养 | 培养、积蓄 | 化养→在积攒 |

**常用五行四柱速查**：

| 五行 | 长生 | 帝旺 | 墓 | 绝 |
|------|------|------|----|----|
| 金 | 巳 | 酉 | 丑 | 寅 |
| 木 | 亥 | 卯 | 未 | 申 |
| 水 | 申 | 子 | 辰 | 巳 |
| 火 | 寅 | 午 | 戌 | 亥 |
| 土 | 申 | 子 | 辰 | 巳 |

> 用法：动爻申金→化出巳火，金长生在巳→「化长生」，后劲足。
> 动爻寅木→化出申金，木绝在申→「化绝」，有始无终。

## 十、伏神旁注要点（第三步·附用）

> 此节为 SKILL.md 第三步·附（伏神旁注）的详细规则。当用神明现但其爻下伏有他神时触发。

| 伏神与用神关系 | 一句话旁注 | 第七步权重 |
|---------------|-----------|-----------|
| 伏神克用神 | 对方受制于某隐因 | 中 — 需在综合断语中点出 |
| 用神克伏神 | 对方在压制某问题 | 低 — 旁注即可 |
| 伏神生用神 | 对方有隐而未用的后援 | 中 — 注意时效 |
| 用神生伏神 | 对方被某隐事消耗 | 高 — 同「伏去生飞泄气」 |
| 比和 | 对方有平行的隐藏面向 | 低 — 点到为止 |

**触发路径**：伏神旁注不改变用神旺衰判定（那是第二步的事），但**第七步综合断语必须引用**此旁注。在综合断语中其权重不低于六兽提点。

**与第3.5步的区别**：
- 第3.5步：用神**不现于卦面** → 查伏神代用神，执行完整飞伏分析
- 第三步·附：用神**明现**但其下伏他神 → 仅做一句话级别的旁注，不替代用神
