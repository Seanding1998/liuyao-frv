# 特殊格局速查

## 用神相关格局

| 格局 | 条件 | 含义 |
|------|------|------|
| 用神持世 | 世爻即用神 | 事在掌握，吉 |
| 用神两现 | 卦中有两个用神 | 事有分叉，取旺相者为主 |
| 用神不现 | 卦中无用神 | 需查伏神（见 fushi-riyue-guashen.md） |
| 用神两动 | 两个动爻都是用神 | 事路两条，取与世爻相应者 |
| 忌神持世 | 世爻为忌神 | 自身是阻碍根源 |

## 动变化出格局

| 格局 | 条件 | 含义 |
|------|------|------|
| 回头克 | 动爻被变爻克制 | 原计划受阻 |

## 卦体结构格局

| 格局 | 条件 | 含义 |
|------|------|------|
| 六合卦 | 本卦为六合卦 | 事多牵连，不易速成 |
| 六冲卦 | 本卦为六冲卦 | 事多散乱，吉凶加速 |
| 六合变六冲 | 先合后冲 | 先成后败，先聚后散 |
| 六冲变六合 | 先冲后合 | 先散后聚，历难而成 |
| 反吟 | 变卦地支与本卦相冲 | 事有反复 |
| 伏吟 | 变卦地支与本卦相同 | 痛苦呻吟，停滞 |
| 独发 | 仅一爻发动 | 焦点明确，此爻是卦眼 |
| 独静 | 仅一爻不发动 | 处处变动，独此不变 |

## 常见六合卦（共8卦，依纳支六合判定）

| 宫位 | 六合卦 |
|------|--------|
| 乾宫 | 天地否 |
| 兑宫 | 泽水困 |
| 离宫 | 火山旅 |
| 震宫 | 雷地豫 |
| 坎宫 | 水泽节 |
| 艮宫 | 山火贲 |
| 坤宫 | 地雷复、地天泰 |

> 注：六合卦是卦内初四/二五/三上爻地支两两相合的结构属性，仅以上8卦满足。雷泽归妹、风天小畜等虽名带"合"意，但纳支不合，非六合卦。

## 常见六冲卦（共10卦）

八纯卦（乾为天、坤为地、坎为水、艮为山、震为雷、巽为风、离为火、兑为泽）均为六冲卦，加上：
- 巽宫：天雷无妄
- 坤宫：雷天大壮

> 注：六冲卦是卦内初四/二五/三上爻地支两两相冲的结构属性。天火同人、火水未济、雷山小过、水天需等虽含冲象，但纳支不全冲，非六冲卦。
>
> 🚨 **常见误判陷阱**：以下卦名虽含「雷」「风」「水」「火」等字，但**不是**六冲卦，Agent 禁止凭卦名字面或手动对冲计算将其标记为六冲卦：
> - 风雷益（☴☳）— 巽宫三世卦，非六冲
> - 风火家人（☴☲）— 巽宫二世卦，非六冲
> - 水雷屯（☵☳）— 坎宫二世卦，非六冲
> - 火雷噬嗑（☲☳）— 巽宫五世卦，非六冲
> - 山雷颐（☶☳）— 巽宫游魂卦，非六冲
> - 泽雷随（☱☳）— 震宫归魂卦，非六冲
>
> ⛔ **解卦时以 paipan.py 的 `patterns.ben_liuchong_gua` 为准，禁止手动计算上下卦对冲关系来替代 JSON 判定。**
