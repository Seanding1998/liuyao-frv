# 六爻解卦分析 Skill

> 专业六爻纳甲筮法排盘 + 解卦工具 —— 自动摇卦排盘 → 十步法逐层分析 → 可交付的 HTML 卦象报告。

## 这是什么？

一个基于传统六爻纳甲筮法的 AI 排盘与解卦 Skill。只需告诉它你想占卜的事，Skill 自动完成：

1. **智能排盘**：分析意图 → 三币摇卦 → sxtwl 四柱计算 → 定卦/世应/六神/伏神
2. **九步解卦**（十步法 = 第零步排盘 + 第一至九步解卦）：按严谨流程逐层分析每一步，最终输出：

1. **终端文本报告**：每一步的分析过程与最终断语，半文半白、有理有据
2. **HTML 可交付报告**：一份排版美观、可直接发送给问卦人的网页版卦象报告

## 适用场景

- 想占卜某事，但不会手动排盘——只需说出问题，Skill 自动完成摇卦排盘 + 解卦全流程
- 已有排盘数据，只需要专业解卦——可直接传入卦象 JSON
- 需要生成可保存、可打印、可分享的卦象报告
- 对六爻有一定了解，希望看到严谨的推导过程

## 使用方式

### 方式一：自动排盘（推荐）

直接告诉 Agent 你想占卜的事，Skill 自动完成全流程：

```
用户: 我想看看下个月的工作运势
Agent: 分析你的问题，意图类别为「官运」，主用神为「官鬼」。是否正确？
用户: 是
Agent: [执行三币摇卦 → 排盘 → 输出卦象 → 进入九步解卦]
```

> 首次使用需确保 `sxtwl` 已安装：`pip install sxtwl`

### 方式二：传入已有排盘数据

如果你已有排盘结果，可提供 JSON 数据直接进入解卦流程：

| 字段 | 说明 | 示例 |
|------|------|------|
| 占卦时间 | 公历/农历时间 | 2026年5月21日13:54 |
| 四柱 | 年月日时干支 | 丙午 癸巳 乙未 癸未 |
| 月建 | 当前月建地支 | 巳 |
| 日辰 | 占卦日日支 | 未 |
| 旬空 | 日柱旬空地支 | 辰巳 |
| 所问之事 | 问卦人的问题 | "午未申三个月的工作运势" |
| 意图类别 | 求财/官运/学业/感情/健康/孕产/出行/失物/词讼/天气/通用 | 官运 |

**每个爻的数据**：爻位、地支、五行、六亲、世应标记、是否发动、变爻信息、六兽、旬空等。

## 输出内容

### 终端报告（十步完整流程）

严格按十步流程依次输出，每一步有强制输出格式：

0. **自动排盘** — 提问→意图分析→三币摇卦→四柱排盘→JSON 卦象
1. **审题取用神** — 确定分析核心（官鬼/妻财/父母……）
2. **判定用神旺衰** — 月建、日辰生克综合判定
3. **追踪动变路径** — 每一动爻的来龙去脉，特殊格局检测
3.5 **伏神分析**（如用神不现）— 飞伏生克与出伏时机
4. **分析世应关系** — 人我格局
5. **六兽取象** — 神煞修饰
6. **应期推断** — 时间窗口推算
7. **综合断语** — 定性总结 + 建议
8. **断后校验** — 交叉验证 + 16条原则逐条检查

### HTML 报告

第九步校验通过后，自动生成一份独立的 HTML 文件，包含：
- 卦象标题与基本信息
- 六爻盘面可视化
- 各步骤分析摘要
- 综合断语与应期建议
- 校验印记

HTML 报告使用内联 CSS，无需外部依赖，可直接在浏览器打开、打印或分享。

## 特色

- **严谨不敷衍**：每一步有门禁、有强制输出、有检查点，不跳步、不合并
- **可追溯**：每条结论可回溯到具体分析步骤
- **不自欺**：第八步断后自动校验，旺衰/动变/世应/应期交叉验证后，第九步生成 HTML 报告
- **凶中见救**：凶兆不渲染恐惧，指出化解方向
- **可交付**：HTML 报告排版美观，适合正式场景交付

## 限制

- **非迷信工具**：六爻是一种传统决策辅助工具，不替代理性判断
